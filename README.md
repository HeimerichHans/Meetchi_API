# Meetchi_API
API for Meetchi

Initialisez Node:
```cmd
npm init
```

Installez les paquets nécessaires: 
```cmd
npm istall
```

Initialisez l'ORM prisma:
```cmd
npx prisma init
```

Dans le fichier **`.env`**, configurez la variable **`DATABASE_URL`**

Récupérez l'architecture de la base de donnée:
```cmd
npx db pull
```

Génerez l'ORM:
```cmd
npx prisma generate
```

Démarrez le serveur:
```cmd
node index.js
```
